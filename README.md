# Visualiser for Cyton, experimental
## requires node.js, python3
* Start Cyton emulator
    * `npm install`
    * `node index.js`
* Start the visualisation
    * `python3 -m pip install pyqt5`
    * `python3 setup.py install`
    * `python3 main.py`
